# Assets Directory

This directory should contain the following image files:

## Required Images

1. **icon.png** (1024x1024)
   - Main app icon
   - Should be a square PNG with no transparency

2. **splash.png** (1242x2436)
   - Splash screen image
   - Will be shown while the app is loading

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Only the center 66% will be visible

4. **favicon.png** (48x48 or larger)
   - Web favicon

## Generating Icons

You can use these tools to generate all required sizes:
- https://appicon.co/
- https://www.appicon.build/
- https://makeappicon.com/

## Temporary Placeholders

For now, you can use simple colored squares as placeholders:
- Use any image editor to create solid color PNG files
- Or download free icon templates from the resources above
